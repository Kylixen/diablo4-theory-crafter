# Knowledge

Evergreen game reference — not season-partitioned. This is the shared
knowledge that builds in [`characters/`](../characters/) draw from.

- [`mechanics/`](mechanics/) — game systems (resistances, damage buckets,
  crowd control, etc.)
- [`items/`](items/) — uniques, aspects, and other gear
- [`skills/`](skills/) — class skills, glyphs, paragon boards

Edited in place as the game changes; git history is the record of what
changed and when a mechanic was reworked.
